### 👋 Hey, I'm Saurabh

By day, a **Product Manager** building AI software at scale — currently **Partner PM on Microsoft AI (Copilot)**, formerly Google. By night and weekend, a **tinkerer** who turns ideas into working tools.

I code just enough to be dangerous: a PM who can ship a prototype, debug an API call, and occasionally break things in interesting ways. **[Sticky Wicket Labs](https://stickywicketlabs.com)** is my playground — weekend builds, no roadmaps, no committee reviews. Just ideas, code, and that satisfying moment when it all comes together.

> 🏏 **My most important user is four years old.** He might only want bedtime stories for a few more years — so I'm building tools that put him on the page, saving the day alongside the Hulk. That's a user need worth losing sleep over.

### 🧭 How I Think About Product

```text
1. Find a user need worth losing sleep over — then obsess over it.
2. Ship the prototype. "Try it" beats "what if" every time.
3. Build it exactly the way it should work. No compromises, no committee.
4. Learn in public. Every project should teach you something —
   some teach you many things by breaking in interesting ways.
5. The best part of making things is the people you meet along the way.
```
